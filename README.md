# gazeta-vybor-media-scraper

Скрипт для автоматической загрузки изображений со страниц новостного портала "Газета Выбор".

## Описание

Этот инструмент использует Playwright для автоматизации браузера, скроллинга страницы до конца, извлечения URL-адресов изображений из галереи новостных статей и загрузки их локально.

## Требования

- Python 3.14+
- Установленный и настроенный Playwright

## Установка

```bash
# Установка зависимостей
uv sync

# Установка браузеров Playwright
uv run playwright install chromium
```

## Использование

```bash
uv run main.py <URL> [OPTIONS]
```

### Аргументы

- `url` — URL страницы с новостью/галереей изображений (обязательный)

### Опции

- `--dir` — Директория для сохранения изображений (по умолчанию: `downloaded_images`)
- `--concurrent` — Количество одновременных загрузок (по умолчанию: 10)

### Примеры

```bash
# Загрузка изображений в папку по умолчанию
uv run main.py https://example.com/news/123

# Загрузка в указанную директорию
uv run main.py https://example.com/news/123 --dir ./images

# Увеличение числа одновременных загрузок
uv run main.py https://example.com/news/123 --concurrent 20
```

## Как это работает

1. Запускается headless-браузер Chromium
2. Переход на указанный URL
3. Автоматический скроллинг страницы до конца для подгрузки всех изображений
4. Извлечение URL-адресов изображений из HTML-элементов `.swiper-container-thumbs .swiper-slide img`
5. Параллельная загрузка изображений с сохранением оригинальных имен файлов
